1. runClassifJTrad
------------------

	Faz uma classificação usando como hipótese uma sigmóide aplicada a uma função linear com uma variável. A função custo utilizada é a tradicional, e não a utilizada na regressão logistica. O objetivo é mostrar que a hipótese, quando usada dessa forma, é uma função não convexa, o que significa que o gradiente não poderia ser aplicado diretamente (poderia cair em mínimos locais).
